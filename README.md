# Exercises-for-Course-Nim_Tutorial-of-Steves_teacher

# Nim — Practice Exercises on CodeWars

A structured set of CodeWars kata collections for the [Nim Programming Tutorial](#).
56 lessons · 7 blocks · two collections per lesson.

| Collection | What it covers | Difficulty |
|------------|----------------|------------|
| **Core** | New constructs introduced in the lesson | 8 kyu |
| **Extra** | New constructs combined with previously learned material | 7–8 kyu |

> **—** marks lessons whose constructs cannot be tested in a CodeWars kata
> (stdin, filesystem, async, threads, terminal, package manager, FFI).
